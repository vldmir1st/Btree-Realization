# Btree-Realization
Лаба по структурам

Реализация класса мультимножества на основе Б-дерева (обход постфиксный):
1) Конструктор
2) Конструктор копирования
3) Оператор присваивания
4) Добавление ключа
5) Удаление ключа (замена на максимальный)
6) Поиск ключа
7) Печать дерева
8) Обход дерева
9) Перегрузка оператора + (Объединение)
